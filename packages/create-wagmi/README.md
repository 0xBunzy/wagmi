# create-wagmi

Get up and running quickly with [Wagmi](https://rc.wagmi.sh) by using the `create-wagmi` CLI.

## Installation

```bash
npm init wagmi
# or
pnpm create wagmi
# or
yarn create wagmi
```

## Documentation

For documentation and guides, visit [wagmi.sh](https://rc.wagmi.sh/cli/create-wagmi).
