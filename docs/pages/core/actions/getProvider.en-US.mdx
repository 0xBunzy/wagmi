# getProvider

Action for accessing viem's [Public Client](https://viem.sh/docs/clients/public.html).

```ts
import { getProvider } from '@wagmi/core'
```

## Usage

```ts
import { getProvider } from '@wagmi/core'

const provider = getProvider()
```

## Return Value

```ts
Provider
```

## Configuration

### chainId (optional)

Force a specific chain id. The wagmi `Client`'s `provider` must be set up as a [chain-aware function](/core/client#provider-optional) for this to work correctly.

```ts {5}
import { getProvider } from '@wagmi/core'

const provider = getProvider({
  chainId: 1,
})
```
