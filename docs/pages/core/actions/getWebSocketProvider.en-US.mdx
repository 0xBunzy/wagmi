# getWebSocketProvider

Action for accessing viem's WebSocket [Public Client](https://viem.sh/docs/clients/public.html).

```ts
import { getWebSocketProvider } from '@wagmi/core'
```

## Usage

```ts
import { getWebSocketProvider } from '@wagmi/core'

const webSocketProvider = getWebSocketProvider()
```

## Return Value

```ts
WebSocketProvider
```

## Configuration

### chainId (optional)

Force a specific chain id. The wagmi `Client`'s `webSocketProvider` must be set up as a [chain-aware function](/core/client#websocketprovider-optional) for this to work correctly.

```ts {5}
import { getWebSocketProvider } from '@wagmi/core'

const webSocketProvider = getWebSocketProvider({
  chainId: 1,
})
```
