# useWebSocketProvider

Hook for accessing viem's WebSocket [Public Client](https://viem.sh/docs/clients/public.html).

```ts
import { useWebSocketProvider } from 'wagmi'
```

## Usage

```tsx
import { useWebSocketProvider } from 'wagmi'

function App() {
  const webSocketProvider = useWebSocketProvider()
}
```

## Return Value

```tsx
WebSocketProvider
```

## Configuration

### chainId (optional)

Force a specific chain id. The wagmi `Client`'s `webSocketProvider` must be set up as a [chain-aware function](/react/client#websocketprovider-optional) for this to work correctly.

```tsx {5}
import { useWebSocketProvider } from 'wagmi'

function App() {
  const webSocketProvider = useWebSocketProvider({
    chainId: 1,
  })
}
```
