# useReconnect

Hook for reconnecting [connectors](/core/connectors).

## Import

```ts
import { useReconnect } from 'wagmi'
```

## Usage

::: code-group
```tsx [index.tsx]
import { useReconnect } from 'wagmi'

function App() {
  const { reconnect } = useReconnect()
}
```
