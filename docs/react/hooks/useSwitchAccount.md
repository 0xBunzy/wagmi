# useSwitchAccount

Hook for switching accounts with [connectors](/core/connectors).

## Import

```ts
import { useSwitchAccount } from 'wagmi'
```

## Usage

::: code-group
```tsx [index.tsx]
import { useSwitchAccount } from 'wagmi'

function App() {
  const { switchAccount } = useSwitchAccount()
}
```
